Xpepemod Documentation

Welcome to the Xpepemod documentation. Xpepemod is a modding framework for Xworm v3.1 that provides theming, event handling, and logging capabilities.

Getting Started

To get started with Xpepemod:

  1. Add a reference to Xpepemod.dll in your project
  2. Create a class that implements the Xpepemod.IMod interface
  3. Implement the OnLoad() method
  4. Build your project and place the DLL in the Mods folder

See the API documentation for detailed examples and usage instructions.

Example Mod

The solution includes an ExampleMod project that demonstrates how to create a mod for Xpepemod.

The ExampleMod:

  • Uses the logging system with different levels
  • Applies a dark theme to the application
  • Adds a menu item to the context menu

Check out the source code in the ExampleMod directory for a complete example.
